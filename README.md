### Todo_Project_Django

```
# How to Run Server
py manage.py runserver          # https://127.0.0.1:8000/ or https://localhost:8000    
# How to Change Port
py manage.py runserver 8888     # https://127.0.0.1:8888/ or https://localhost:8888  
# How to Stop Server
ctrl+c

```
